# Hot Prospects 🤝

**Hot Prospects** — это приложение для сбора и управления контактными данными на конференциях и мероприятиях. Оно позволяет мгновенно обмениваться информацией с помощью QR-кодов, разделять контакты на «связанные» и «ожидающие», а также настраивать напоминания для последующего общения.

## ✨ Особенности

- **QR-технологии**: 
    - Генерация персонального QR-кода на основе ваших данных для быстрого шеринга.
    - Встроенный сканер для мгновенного добавления новых «проспектов» в список.
- **Умное управление данными**:
    - Использование **SwiftData** для надежного хранения контактов.
    - Фильтрация по статусу связи: Все, Связанные (Contacted), Несвязанные (Uncontacted).
    - Динамическая сортировка по имени или по времени последнего контакта.
- **Локальные уведомления**: Возможность запланировать напоминание («Follow up»), чтобы не забыть связаться с человеком позже.
- **Интерактивный интерфейс**: 
    - Кастомные свайп-действия (Swipe Actions) для изменения статуса контакта.
    - Контекстные меню для быстрого взаимодействия.
- **Работа с буфером обмена**: Копирование email-адреса одним долгим нажатием.

## 🛠 Технологии

- **Swift 5.9+ / SwiftUI**
- **SwiftData**: Для управления базой данных и связей.
- **CoreImage**: Генерация QR-кодов высокого качества.
- **UserNotifications**: Настройка локальных напоминаний.
- **Environment & State**: Управление глобальным состоянием приложения.

## 🚀 Установка

1. Склонируйте репозиторий:
    ```bash
    git clone https://github.com/Serge-17/Hot-Prospects.git
    ```
2. Откройте `Hot Prospects.xcodeproj` в Xcode.
3. **Важно**: Для полноценного тестирования сканирования лучше использовать реальное устройство (требуется доступ к камере).
4. Запустите проект на iOS 17.0 или выше.
